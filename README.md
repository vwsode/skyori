# WEBPACK

## Команды

### Запуск json-server

yarn server

### Запуск сервера для разработки

```shell
yarn run start
```

### Сборка проекта без оптимизации

```shell
yarn run build-dev
```

### Сборка проекта с оптимизацией

```shell
yarn run build-prod
```

### Очистка папки dist

```shell
yarn run clear
```
